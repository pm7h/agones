{{< /blocks/cover >}} {{% blocks/lead color="primary" %}} What is Agones? Agones is an open source project that is built to be a batteries-included, dedicated game server scaling and orchestration platform. It is built as a native extension of [Kubernetes](https://kubernetes.io), with the flexibility you need to tailor it to the requirements of your multiplayer game. Anywhere you can run Kubernetes, you can run Agones. {{% /blocks/lead %}} {{< blocks/section color="dark" >}} {{% blocks/feature icon="fas fa-file-code" title="Fleets of GameServers" url="./docs/getting-started/" %}} Define Fleets of GameServers, autoscaling and more, through YAML or the Kubernetes API. {{% /blocks/feature %}} {{% blocks/feature icon="fas fa-wrench" title="Integrated SDK" url="./docs/guides/client-sdks/" %}} Integrated SDK for managing game server lifecyle, health and configuration.
